# GUÍA RÁPIDA DE USO

## 🚀 Inicio Rápido

### Primera Vez
1. Abre `index.html` en tu navegador
2. Configura tu capital inicial
3. Comienza a agregar clientes y préstamos

### Flujo Básico
```
Cliente → Préstamo → Registrar Pagos → Ver Dashboard
```

## 📋 Tareas Comunes

### Agregar Cliente
1. Clientes → + Nuevo Cliente
2. Llenar datos
3. Guardar

### Crear Préstamo
1. Préstamos → + Nuevo Préstamo
2. Seleccionar cliente
3. Configurar monto y cuotas
4. Guardar

### Registrar Pago
1. Buscar préstamo en lista
2. Click "Registrar Pago"
3. Ingresar monto y fecha
4. Guardar

### Consultar Información
- **Dashboard:** Métricas generales
- **Análisis:** Rentabilidad y top clientes
- **Simulador:** Proyecciones de crecimiento
- **Asistente:** Preguntas y recomendaciones

## ⚡ Atajos de Teclado

- `Ctrl+K` - Buscar cliente
- `Ctrl+N` - Nuevo cliente (en sección Clientes)

## 💡 Consejos

1. **Revisa diariamente el Dashboard** para ver pagos pendientes
2. **Monitorea el score** de tus clientes
3. **Usa el Simulador** para planificar tu crecimiento
4. **Consulta al Asistente** para insights rápidos
5. **Mantén actualizado** el registro de pagos

## 🔧 Mantenimiento

- Los datos se guardan automáticamente en tu navegador
- No necesitas hacer nada especial para guardar
- Si cambias de dispositivo, los datos NO se transfieren
- Considera exportar backups periódicamente (función futura)

## ⚠️ Importante

- **No borres los datos del navegador** o perderás la información
- **Usa un navegador moderno** (Chrome, Firefox, Edge, Safari)
- **Mantén backup** de tus datos importantes
