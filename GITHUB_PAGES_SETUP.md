## 🚀 Pasos para Activar GitHub Pages

### 1. Ve a la Configuración de Pages
1. Abre tu repositorio: https://github.com/braianruaimi/MindSetCapital
2. Click en **Settings** (Configuración)
3. En el menú lateral, busca **Pages**

### 2. Configura la Fuente
- **Source (Fuente)**: Deploy from a branch
- **Branch**: `main`
- **Folder**: `/ (root)`
- Click en **Save**

### 3. Espera el Despliegue
- GitHub procesará tu sitio (toma 1-3 minutos)
- Verás un mensaje: "Your site is live at..."

## 🌐 URL de tu Aplicación

Una vez configurado, tu app estará disponible en:

```
https://braianruaimi.github.io/MindSetCapital/
```

## 📱 Instalación en Móvil

### Android (Chrome/Edge):
1. Abre la URL en el navegador
2. Toca el menú (⋮) > "Agregar a pantalla de inicio"
3. La app se instalará como PWA

### iOS (Safari):
1. Abre la URL en Safari
2. Toca el botón de compartir 
3. Selecciona "Agregar a pantalla de inicio"
